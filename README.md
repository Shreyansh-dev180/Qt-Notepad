# 📝 Notepad v1

A simple and lightweight Notepad application built with **C++** and the **Qt Framework**.

This project was created as a learning project while exploring Qt and desktop application development. Inspired by Microsoft Notepad, this first version focuses on simplicity and ease of use.

## ✨ Features

* Create a new page.
* Save notes as `.txt` files.
* Clean and minimal user interface.
* Native Windows file dialog support.
* Built with C++ and Qt.

## 🚀 Installation and Usage

### Method 1: Clone the Repository

Clone the repository:

```bash
git clone https://github.com/Shreyansh-dev180/Qt-Notepad.git
```

Open the cloned folder and launch:

```text
Notepad.exe
```

Enjoy using the application!

---

### Method 2: Download the Release ZIP

1. Download the ZIP file from the Releases section.
2. Extract the ZIP file.
3. Open the extracted folder.
4. Launch:

```text
Notepad.exe
```

That's it! No installation is required.
